# Unit 2 Group Project for Code.org HTML and CSS Unit

A multi-page website about ancient civilizations (Ancient China, Ancient Egypt, and Ancient Mesopotamia), built with HTML and CSS as part of Code.org's HTML and CSS unit.

**Final Grade: 92**

> **Note for students doing this project:** Make sure to attribute authors of photos on the page instead of just in the alternate text.

All code requires the photos that are included in this repository (as of 12/17/25).

## Pages

- `index.html` – Homepage
- `china.html` – Ancient China
- `egypt.html` – Ancient Egypt
- `meso.html` – Ancient Mesopotamia
- `colors.html` – Ancient Colors
- `colors.css` – Stylesheet
